Git Class 03

Task:
Track your project using Git.

	1. Decide where you will create the repository.
	2. Navigate the terminal to the right location.
	3. Initialize the empty git repo --> git init
	4. git status --> untracked files -- red color
	5. Add all the files in the staging area (so that gits starts tracking them)-- git add .
	6. Add all the files in the repository --> git commit -m "commit message"
